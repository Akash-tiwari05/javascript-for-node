# 🟨 Hands-On JavaScript (Before Moving to Node.js)

This repository contains my **hands-on JavaScript practice** focused on building a **strong foundation before starting Node.js**.  
The goal is to deeply understand **core JavaScript concepts**, **logic building**, and **real-world patterns** that are essential for backend development.

---

## 🎯 Why This Repository?

Before jumping into **Node.js**, it is important to:
- Understand how JavaScript works internally
- Be comfortable with async behavior
- Write clean and predictable logic
- Avoid treating Node.js as “magic”

This repo helps me bridge the gap between **basic JS** and **server-side JavaScript (Node.js)**.

---

## 🧠 Topics Covered

### 1️⃣ JavaScript Fundamentals
- Variables (`var`, `let`, `const`)
- Data Types (Primitive & Non-Primitive)
- Operators
- Type Coercion
- Truthy & Falsy values

### 2️⃣ Control Flow
- `if / else`
- `switch`
- Loops (`for`, `while`, `do-while`)
- Break & Continue

### 3️⃣ Functions
- Function Declarations
- Function Expressions
- Arrow Functions
- Parameters vs Arguments
- Default Parameters
- Return vs Console.log

### 4️⃣ Scope & Hoisting
- Global Scope
- Function Scope
- Block Scope
- Hoisting behavior of `var`, `let`, `const`
- Execution Context (Basics)

### 5️⃣ Arrays & Objects
- Array methods (`map`, `filter`, `reduce`, `forEach`)
- Object creation & access
- Nested objects
- Shallow vs Deep copy
- Destructuring

### 6️⃣ JavaScript Internals (Very Important)
- Call Stack
- Memory Heap
- Execution Context
- Event Loop (Basic understanding)
- Synchronous vs Asynchronous JS

### 7️⃣ Asynchronous JavaScript
- Callbacks
- Callback Hell
- Promises
- `async / await`
- Error handling (`try / catch`)

### 8️⃣ Closures & Higher-Order Functions
- Closure use cases
- Lexical scope
- Functions returning functions
- Real-world examples

### 9️⃣ This Keyword
- `this` in:
  - Global scope
  - Functions
  - Objects
  - Arrow functions
- Common interview pitfalls

### 🔟 ES6+ Features
- Spread & Rest operators
- Template literals
- Optional chaining
- Nullish coalescing
- Modules (`import / export`)
